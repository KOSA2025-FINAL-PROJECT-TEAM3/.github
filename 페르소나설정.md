## 페르소나 개요

뭐냑?는 **양면 시장(Two-sided Market)** 플랫폼입니다:

| 사용자 그룹 | 역할 | 주요 니즈 |
| --- | --- | --- |
| **시니어** | 약 복용 대상자 | 간편한 약 관리, 복용 알림 |
| **자녀/보호자** | 원격 관리자 | 부모님 건강 모니터링, 안심 |
| **의사** | 전문 상담 제공자 | 복약 지도, 상호작용 경고 |

## 주요 페르소나

### 1. 시니어 (부모님)

### 👵 페르소나 A: "김순자" (88세)

**기본 정보**

- **나이**: 88세
- **거주지**: 경기도 수원시 (단독 주택)
- **가족**: 배우자 사망, 자녀 2명 (서울, 부산 거주)
- **직업**: 은퇴 (전 초등학교 교사)
- **건강 상태**: 고혈압, 당뇨병, 골다공증

**배경 이야기**

> "남편이 돌아가신 후 10년째 혼자 살고 있어요. 자식들이 주말에 가끔 오지만 평일엔 혼자에요. 약을 여러 개 먹어야 하는데 자꾸 까먹어요. 약봉지에 쓰여있는 글씨도 잘 안 보이고요."
> 

**Pain Points (불편 사항)**

- 🔴 **약 복용 시간을 자주 잊음** - 하루 3번 복용 약을 아침에만 먹거나 건너뜀
- 🔴 **여러 약 관리가 복잡함** - 혈압약, 당뇨약, 칼슘제, 영양제 등 5가지 이상
- 🔴 **약 이름이 헷갈림** - "아모잘탄", "메트포르민" 같은 전문 용어 이해 어려움
- 🔴 **자녀들이 걱정함** - "엄마, 약 먹었어?" 전화가 부담스러움
- 🔴 **약국 방문이 힘듦** - 무릎 관절염으로 먼 거리 이동 곤란
- 🔴 **스마트폰 사용 서툼** - 복잡한 앱은 사용 포기

**목표 & 니즈**

- ✅ 약 먹는 시간에 알림 받기 (크고 명확하게)
- ✅ 버튼 하나로 "약 먹었음" 체크
- ✅ 자녀들이 안심할 수 있게 내 복용 기록 공유
- ✅ 약 이름 대신 사진으로 약 구분
- ✅ 궁금한 건 의사에게 쉽게 물어보기

**기술 수준**

- 📱 스마트폰: 전화, 카카오톡, 사진 촬영 가능
- 🚫 인터넷 검색, 앱 설치, 복잡한 설정 어려움
- ✅ **큰 글씨, 단순한 버튼 선호**

**하루 일과**

```
06:00 - 기상
06:30 - 아침 식사 후 약 복용 (고혈압약, 당뇨약)
12:00 - 점심 식사
18:00 - 저녁 식사 후 약 복용 (칼슘제)
21:00 - 취침 전 약 복용 (수면제)
22:00 - 취침

```

**주요 사용 시나리오**

1. **아침 7시**: 스마트폰 알림 울림 → "아침 약 먹을 시간이에요 💊"
2. **약 복용 후**: 큰 버튼 터치 → "✓ 약 먹었어요" (자녀에게 자동 알림)
3. **약봉지 받음**: 사진 촬영 → OCR 자동 인식 → 약 정보 등록
4. **궁금한 점**: 의사 채팅 → "이 약은 언제 먹어야 하나요?"

### 👴 페르소나 B: "박영수" (68세)

**기본 정보**

- **나이**: 68세
- **거주지**: 서울 은평구 (아파트)
- **가족**: 배우자 있음, 자녀 1명 (미국 거주)
- **직업**: 은퇴 (전 중소기업 임원)
- **건강 상태**: 협심증, 고지혈증, 전립선 비대증

**배경 이야기**

> "퇴직 후 건강이 안 좋아져서 약을 많이 먹게 됐어요. 딸이 미국에 있어서 자주 못 만나는데, 건강 때문에 걱정을 끼치기 싫어요. 스마트폰은 좀 쓸 줄 아는데, 앱이 너무 복잡하면 못 써요."
> 

**Pain Points**

- 🔴 **약-약 상호작용 우려** - 여러 병원에서 처방받아 약이 충돌할까 걱정
- 🔴 **복용 기록 관리 어려움** - 언제 먹었는지 기억 안 남
- 🔴 **해외 자녀와 소통 어려움** - 시차 때문에 전화하기 힘듦
- 🔴 **약 부작용 판단 어려움** - 어지러운 게 약 때문인지 모르겠음

**목표 & 니즈**

- ✅ 복용 기록 자동 저장 (나중에 병원 갈 때 보여주기)
- ✅ 딸이 원격으로 내 건강 확인
- ✅ 약 충돌 자동 체크
- ✅ 복약 리포트 생성 (PDF로 의사에게 제출)

**기술 수준**

- 📱 스마트폰: 뉴스 앱, 유튜브, 인터넷 검색 가능
- ✅ 앱 설치 및 기본 설정 가능
- ⚠️ 복잡한 UI는 학습 필요

**주요 사용 시나리오**

1. **병원 방문 전**: "지난 한 달 복약 리포트 보기" → PDF 다운로드 → 의사에게 제출
2. **새 약 처방받음**: 약 등록 → 기존 약과 충돌 체크 → "⚠️ 와파린과 아스피린 동시 복용 주의"
3. **해외 자녀**: 미국에 있는 딸이 앱으로 아버지 복용 현황 실시간 확인

### 2. 자녀/보호자

### 👨‍💼 페르소나 C: "이민준" (45세)

**기본 정보**

- **나이**: 45세
- **거주지**: 서울 강남구
- **가족**: 배우자, 자녀 2명 (초등학생), 부모님(수원 거주)
- **직업**: IT 기업 과장 (주 6일 근무)
- **소득**: 중상층

**배경 이야기**

> "어머니가 혼자 사시는데 당뇨랑 고혈압이 있으세요. 주말에 찾아뵙지만, 평일엔 제대로 약을 드시는지 걱정돼요. 퇴근하고 전화드려도 '먹었어, 먹었어' 하시는데 사실은 잘 모르시는 것 같아요. 회사에서도 어머니 생각에 집중이 안 돼요."
> 

**Pain Points**

- 🔴 **평일엔 부모님 방문 불가** - 회사 업무로 주중엔 시간 없음
- 🔴 **전화로 확인하기 애매함** - "약 드셨어요?" 질문이 잔소리처럼 들릴까 부담
- 🔴 **복용 여부 확신 못함** - 어머니가 "먹었다"고 하셔도 의심스러움
- 🔴 **응급 상황 대처 어려움** - 혹시 쓰러지시면 어떡하지?
- 🔴 **약 관리 전문 지식 없음** - 어떤 약이 중요한지 모름

**목표 & 니즈**

- ✅ 부모님 복용 현황 실시간 확인 (출퇴근 지하철에서)
- ✅ 약 안 드셨을 때 즉시 알림 받기
- ✅ 원격으로 약 스케줄 등록/수정
- ✅ 복약 순응도 통계 확인 (월별, 주별)
- ✅ 약 충돌 자동 경고 (새 약 처방받으실 때)
- **✅ 긴급 상황 시 알림 (24시간 이상 미복용)**

**기술 수준**

- 📱 IT 전문가 수준
- ✅ 모든 스마트폰 기능 자유롭게 사용
- ✅ 복잡한 앱도 빠르게 학습

**하루 일과**

```
07:00 - 기상, 출근 준비
08:30 - 출근 (지하철에서 부모님 복용 현황 체크)
09:00~18:00 - 회사 업무
18:30 - 퇴근 중 앱 확인 → "어머니 저녁 약 복용 완료 ✓"
19:00 - 귀가, 가족과 저녁 식사
21:00 - 어머니께 전화 (안부 확인)
22:00 - 취침 전 내일 어머니 약 알림 설정

```

**주요 사용 시나리오**

1. **출근 중**: 앱 열기 → "어머니 아침 약 복용 완료 ✓" 확인 → 안심
2. **점심시간**: 알림 수신 → "⚠️ 어머니가 점심 약을 복용하지 않으셨습니다" → 전화 또는 메시지 발송
3. **주말**: 어머니 방문 → 새로 처방받은 약 사진 촬영 → OCR 자동 등록 → 기존 약과 충돌 체크
4. **월말**: "이번 달 복약 순응도 85%" 확인 → 병원 예약 고려

- **김순자**: 알림 확인 → 약 복용 → "✓ 약 먹었어요" 버튼 터치
- **이민준 앱**: 실시간 알림 수신 → "어머니가 아침 약을 복용하셨습니다 ✓"

### 출근 중 (08:30)

- **이민준**: 지하철에서 앱 확인
- **대시보드**: "어머니 복약 현황 - 오늘 1/3 완료"
- **이민준**: 안심하고 출근

### 점심 (12:00)

- **김순자**: 점심 먹고 약 복용 깜빡 잊음
- **시스템**: 30분 경과 → 알림 재발송
- **김순자**: 알림 보고 약 복용

### 저녁 (18:30)

- **김순자**: 저녁 약 복용 완료
- **이민준**: 퇴근 중 확인 → "오늘 복용 완료 3/3 ✓"
- **이민준**: 안심하고 귀가

### 시나리오 2: 새 약 처방받음

**등장 인물**: 박영수(시니어), 정민우(의사), 박지현(자녀/간호사)

### 병원 방문 후 (14:00)

- **박영수**: 새 약 처방전 받음 → 약국 방문
- **정민우**: 처방전 조제 중 → 뭐냑? 앱으로 기존 복용 약 확인
- **정민우**: "⚠️ 환자분이 현재 와파린을 복용 중이십니다. 새 처방약인 아스피린과 상호작용 주의 필요합니다."
- **정민우**: 의사에게 처방 재확인 요청 → 약 변경

### 병원에서 (14:30)

- **박영수**: 약봉지 받음 → 스마트폰 카메라로 촬영
- **뭐냑? OCR**: 약 정보 자동 인식 → "신규 약 등록 완료"
- **시스템**: 기존 약과 충돌 검사 → "✓ 상호작용 문제 없음"

### 자녀 확인 (15:00)

- **박지현**: 야간 근무 준비 중 앱 확인
- **알림**: "아버지께서 새로운 약을 등록하셨습니다"
- **박지현**: 약 정보 확인 → "고지혈증약 추가, 문제없음" → 안심

### 시나리오 3: 약 미복용 긴급 상황

**등장 인물**: 김순자(시니어), 이민준(자녀)

### 저녁 (19:00)

- **김순자**: 저녁 약 복용 시간 (18:00) 놓침
- **시스템**: 1시간 경과 → 김순자에게 알림 재발송
- **김순자**: 알림 확인 못함 (TV 시청 중)

### 저녁 (19:30)

- **시스템**: 1시간 30분 경과 → 자녀에게 알림 발송
- **이민준**: "⚠️ 어머니가 저녁 약을 복용하지 않으셨습니다"
- **이민준**: 어머니께 전화 → "엄마, 저녁 약 드셨어요?"
- **김순자**: "아, 깜빡했네!" → 전화 끊고 약 복용

### 저녁 (19:35)

- **김순자**: 약 복용 후 "✓ 약 먹었어요" 터치
- **이민준**: 확인 알림 수신 → 안심

### 시나리오 4: 의사 1:1 상담

**등장 인물**: 김순자(시니어), 정민우(의사)

### 오후 (15:00)

- **김순자**: 새로 받은 약 부작용 의심 (어지러움)
- **김순자**: 뭐냑? 앱 → "의사에게 문의하기" → 채팅 입력
- **김순자**: "약 먹고 나서 어지러운데 약 때문인가요?"

### 오후 (15:30)

- **정민우**: 손님 응대 중 → 스마트폰으로 알림 확인
- **정민우**: 점심시간에 답변 예정

### 점심시간 (12:30 다음날)

- **정민우**: 약국 조용한 시간 → 채팅 답변
- **정민우**: "혈압약의 부작용일 수 있습니다. 증상이 계속되면 병원 방문 권장드립니다. 약 복용 시간을 식후로 조정해보세요."
- **김순자**: 답변 확인 → 다음 복용 시 식후로 변경 → 증상 완화
