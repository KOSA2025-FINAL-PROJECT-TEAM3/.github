## 🔧 플러그인 설치 (최초 1회)

### 1단계: Figma Desktop 앱 열기
⚠️ **중요**: 웹 버전이 아닌 Desktop 앱 필수!

### 2단계: 플러그인 Import
1. 메뉴: `Plugins → Development → Import plugin from manifest...`
2. 이 프로젝트의 `figma-plugin/manifest.json` 선택
3. ✅ "뭐냑? JSON Importer" 플러그인이 개발 플러그인 목록에 추가됨

### 3단계: 완료!
이제 `Plugins → Development → 뭐냑? JSON Importer` 클릭하면 바로 실행됩니다.

## 🚀 JSON Import 방법 (3가지)

### 방법 1: 클립보드 자동 감지 (가장 빠름! ⚡)

```bash
# 1. JSON 복사
cat figma-exports/all-screens-with-flows.json | pbcopy  # Mac
cat figma-exports/all-screens-with-flows.json | xclip -selection clipboard  # Linux

# Windows: 파일을 열어서 Ctrl+A → Ctrl+C

# 2. 피그마 플러그인 열기
# Plugins → Development → 뭐냑? JSON Importer

# 3. 자동으로 클립보드에서 JSON 감지! ✨
# 바로 🚀 Import 버튼만 누르면 됨!
```

**특징:**
- 로컬 스토리지에 자동 저장됨
- 다음에 플러그인을 열면 자동으로 로드
- 매번 복사-붙여넣기 할 필요 없음!

---

### 방법 2: 파일 드래그앤드롭

```
1. 피그마 플러그인 열기
2. JSON 파일을 플러그인 창으로 **드래그**
3. 자동으로 로드됨
4. 🚀 Import Multiple 버튼 클릭
```

---

### 방법 3: URL에서 가져오기 (팀 공유에 최적!)

#### GitHub Raw URL 사용
```
1. JSON을 GitHub에 푸시
2. 파일 열기 → "Raw" 버튼 클릭 → URL 복사
   예: https://raw.githubusercontent.com/KOSA2025-FINAL-PROJECT-TEAM3/Front/main/figma-exports/all-screens-with-flows.json

3. 플러그인에서 🌐 "URL에서 가져오기" 클릭
4. URL 입력
5. 자동으로 다운로드 & 로드
6. 🚀 Import 버튼 클릭
```

#### 로컬 HTTP 서버 사용
```bash
# Python으로 간단한 서버 실행
cd figma-exports
python3 -m http.server 8000

# 이제 URL 사용 가능:
# http://localhost:8000/all-screens-with-flows.json
```

#### GitHub Gist 사용
```
1. gist.github.com에서 새 Gist 생성
2. JSON 내용 붙여넣기
3. "Raw" 버튼 클릭하여 URL 복사
```

## ✨ 결과 확인

### 📊 생성되는 화면 (총 8개)
1. **로그인** - 이메일/비밀번호 입력
2. **회원가입** - 사용자 정보 입력
3. **역할 선택** - 시니어 vs 보호자 선택
4. **시니어 대시보드** - 약 복용 체크리스트
5. **보호자 대시보드** - 가족 모니터링
6. **약 관리** - 약 목록 및 관리
7. **약 등록** - 새 약 추가 폼
8. **가족 관리** - 가족 구성원 관리

### 🔗 프로토타입 플로우 (총 15개 화살표)

#### 인증 플로우
```
로그인 → 역할 선택
회원가입 → 역할 선택
로그인 ↔ 회원가입
```

#### 역할 선택 플로우
```
역할 선택 → 시니어 대시보드 (시니어 선택 시)
역할 선택 → 보호자 대시보드 (보호자 선택 시)
```

#### 메인 네비게이션
```
시니어 대시보드 → 약 관리
시니어 대시보드 → 가족 관리
보호자 대시보드 → 약 관리
보호자 대시보드 → 가족 관리
```

#### 약 관리 플로우
```
약 관리 → 약 등록 (+ 약 추가 버튼)
약 등록 → 약 관리 (저장 버튼)
약 등록 → 약 관리 (뒤로가기)
```

#### 홈 복귀
```
약 관리 → 시니어 대시보드 (홈 버튼)
가족 관리 → 시니어 대시보드 (홈 버튼)
```

## 🎨 화살표 확인 방법

**중요:** PNG 이미지는 정적이라서 화살표가 안 보입니다! 피그마 플러그인으로 JSON을 import해야 화살표를 볼 수 있습니다.

### 방법 1: 프로토타입 모드 (추천)

1. 피그마 오른쪽 상단 **프로토타입 탭** 클릭
2. 화면 선택하면 → **파란색 화살표** 보임!
3. 화살표 클릭하면 연결된 화면 확인 가능
4. 트랜지션 효과 변경 가능 (현재: DISSOLVE 0.3초)

### 방법 2: 프레젠테이션 모드

1. 화면 선택 후 **▶️ 재생 버튼** 클릭
2. 실제로 클릭하면서 화면 전환 테스트
3. ESC로 나가기

### 화면 플로우 다이어그램

```
[로그인]
   ↓
[역할 선택]
   ↓
   ├─→ [시니어 대시보드] ─→ [약 관리] ─→ [약 등록]
   │         ↓                   ↑           ↓
   │    [가족 관리] ←────────────┴───────────┘
   │
   └─→ [보호자 대시보드] ─→ [약 관리] ─→ [약 등록]
             ↓                   ↑           ↓
        [가족 관리] ←────────────┴───────────┘
```

### 연결된 플로우 상세

| 시작 화면 | 클릭 요소 | 이동 화면 |
|-----------|-----------|-----------|
| 시니어 대시보드 | 💊 약 관리 | 약 관리 |
| 시니어 대시보드 | 👨‍👩‍👧 가족 | 가족 관리 |
| 보호자 대시보드 | 💊 약 관리 | 약 관리 |
| 보호자 대시보드 | 👨‍👩‍👧 가족 | 가족 관리 |
| 약 관리 | 🏠 홈 | 시니어 대시보드 |
| 가족 관리 | 🏠 홈 | 시니어 대시보드 |

## 🔧 문제 해결

### 플러그인이 안 보임
→ **피그마 Desktop 앱** 사용 중인지 확인 (웹 버전 안됨)

### "Invalid JSON structure"
→ JSON 파일 **전체** 복사했는지 확인
→ 파일 시작: `{` / 파일 끝: `}`

### 화살표가 안 보임
→ **프로토타입 탭**에서 확인 (디자인 탭에서는 안 보임)
→ 화면 선택 후 오른쪽 패널의 "Prototype" 클릭

### "Font not available" 경고
→ [Inter 폰트](https://fonts.google.com/specimen/Inter) 설치 필요
→ 또는 피그마에서 다른 폰트로 변경

### 클립보드 자동 감지가 안 됨
→ JSON을 복사한 후 플러그인 열기
→ 또는 수동으로 붙여넣기 (Ctrl+V)

### URL 다운로드 실패
→ URL이 올바른지 확인 (https:// 또는 http://)
→ CORS 문제일 수 있음 - GitHub Raw URL 사용 권장

## 💡 Tip

### 피그마에서 수정하기
- 모든 화면은 **Auto Layout** 적용됨
- 색상, 간격, 텍스트 자유롭게 수정 가능
- 컴포넌트로 변환하여 재사용 가능

### 화살표 수정하기
- 프로토타입 모드에서 화살표 클릭
- 트랜지션 효과 변경 가능
- 새로운 연결 추가 가능

### Export하기
- 개별 화면 PNG/SVG Export 가능
- 전체 프로토타입 공유 링크 생성 가능

### 키보드 단축키
- `Cmd/Ctrl + Enter` : 빠른 Import
- `Escape` : 플러그인 닫기
